# MediBook

MediBook is a responsive healthcare appointment-booking website developed as
the Group B capstone project for IMT 405 Web Application Development. It allows
patients to select a medical service, choose an available doctor, pick a date
and time, review the appointment cost in Nigerian naira, and receive an instant
booking confirmation.

## Technologies Used

- HTML5 for page structure and semantic content
- CSS3 for styling, layout, and responsive design
- Vanilla JavaScript for form validation, booking interactions, navigation,
  and currency formatting
- Google Fonts (Inter) for typography

## Setup and Installation

This is a static website with no package dependencies or build process.

1. Clone or download the repository.
2. Open the project directory:

   ```bash
   cd "Group B"
   ```

3. Open the project folder in Visual Studio Code.
4. Open the Extensions panel and install **Live Server** by Ritwick Dey.
5. Right-click `index.html` and select **Open with Live Server**.

The website will open automatically in your default browser. Alternatively,
open `index.html` directly in a modern web browser.
